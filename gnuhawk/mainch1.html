Chapter 1
GNUHAWK

GNUHAWK is a software package containing a shared library and Components that enable GNU Radio blocks to be used within the REDHAWK software framework.

1.1 Introduction

GNUHAWK provides reusability of GNU Radio block algorithms and capabilities by making them interoperable with other REDHAWK Components within the REDHAWK software framework without requiring a separate GNU Radio installation.

The GNUHAWK library provides software that enables a GNU Radio block to be integrated into the REDHAWK software framework. This library enables a GNUHAWK Component to instantiate a GNU Radio block, interact with it by using the block’s setters and getters, and utilize the GNU Radio block algorithm by calling the general_work method. GNUHAWK is based on version 3.6.2 of GNU Radio, which is included as part of the GNUHAWK source baseline, therefore a separate build of GNU Radio is NOT required to install and use GNUHAWK. The installed GNUHAWK library and include files are located under $SDRROOT/dom/deps/gnuhawk.

GNUHAWK follows an integration pattern that enables external algorithms from existing GNU Radio blocks to be used within the REDHAWK processing framework with minimal modifications to the algorithm’s original source code. The GNUHAWK library enables standard REDHAWK Components’ Properties to change the state of an underlying GNU Radio block by interacting with the setters and getters of the block. The GNUHAWK Component’s service function mimics the functionality of the GNU Radio gr_block_executor class as well as the necessary requirements for data flow over the BULKIO interfaces. In addition to maintaining SRI and time stamp context for the data flow, the Component provides the marshalling and unmarshalling of data from standard REDHAWK BULKIO interfaces and presents the data to the block’s general_work method.
