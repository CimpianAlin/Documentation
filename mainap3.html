Appendix C
REDHAWK Dependencies

C.1 Dependencies Packaged with REDHAWK

The following packages are included in the REDHAWK dependencies tarball and are required for REDHAWK:

  • libomniEvents2-2.6.2-4
  • libomniEvents2-devel-2.6.2-4
  • libomniORB4.1-4.1.0-0.3
  • libomniORB4.1-devel-4.1.0-0.3
  • libomniORBpy3-3.0-0.2
  • libomniORBpy3-devel-3.0-0.2
  • omniEvents-bootscripts-2.6.2-4
  • omniEvents-debuginfo-2.6.2-4
  • omniEvents-doc-2.6.2-4
  • omniEvents-server-2.6.2-4
  • omniEvents-utils-2.6.2-4
  • omniORB-bootscripts-4.1.0-0.3
  • omniORB-debuginfo-4.1.0-0.3
  • omniORB-doc-4.1.0-0.3
  • omniORB-servers-4.1.0-0.3
  • omniORB-utils-4.1.0-0.3
  • omniORBpy-debuginfo-3.0-0.2
  • omniORBpy-doc-3.0-0.2

The following packages are included in the REDHAWK dependencies tarball for Red Hat/CentOS 5 and are required for REDHAWK when installing on Red Hat/CentOS 5:

  • boost-1.41.0-1
  • boost-devel-1.41.0-1
  • boost-debuginfo-1.41.0-1

The following packages are included in the REDHAWK dependencies tarball but are recommended rather than required for REDHAWK:

  • apache-log4cxx-0.10.0-2
  • apache-log4cxx-devel-0.10.0-2
  • apache-log4cxx-debuginfo-0.10.0-2

C.1.1 Developer Dependencies

Building REDHAWK assets requires several other development dependencies which must be installed in order to compile new components.

The following packages must be installed in order to develop and build new REDHAWK assets.

  • apr-devel \
  • apr-util-devel \
  • autoconf \
  • automake \
  • expat-devel \
  • gcc-c++ \
  • "java-devel >= 1.6 "\
  • libtool \
  • numpy \
  • python-devel

The following dependencies are required for Red Hat/CentOS 5:

  • e2fsprogs-devel \
  • python-uuid

The following dependencies are required for Red Hat/CentOS 6:

  • libuuid-devel \
  • boost-devel
