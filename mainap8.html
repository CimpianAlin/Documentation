Known Compatability Issues

H.1 Compiling REDHAWK and REDHAWK C++ Components in RHEL/CentOS 6.4

Due to a bug in the boost header files at the time RHEL/CentOS 6.4 was released, REDHAWK and REDHAWK C++ projects do not compile using the system-installed gcc version 4.4.7 (https://bugzilla.redhat.com/show_bug.cgi?id=921792). If your system is configured to receive the latest OS updates, performing a yum update resolves this issue. The issue is corrected in boost-1.41.0-17 and later.
