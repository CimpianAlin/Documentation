Installing GNUHAWK

This chapter describes the process of installing the GNUHAWK library and components. Users may install GNUHAWK from source or pre-built rpms.

Note: To install GNUHAWK, users must have root or sudo access.

Dependencies

GNUHAWK has runtime and build dependencies.

Runtime Dependencies

The following packages must be installed to run GNUHAWK.

  • REDHAWK Core Framework 1.8.4
  • REDHAWK BULKIO Libraries 1.8.4
  • FFTW 3.2.1 or newer (This includes both the FFTW and FFTW-devel rpms.)

    For more information about installing REDHAWK, refer to the Installation chapter of the REDHAWK Manual.

Build Dependencies

The following dependencies are required to build the underlying VOLK GNU Radio library.

  • cmake
  • python-cheetah

Location of Installed Files

  • GNUHAWK library is installed in $SDRROOT/dom/deps/gnuhawk/lib.
  • GNU Radio’s VOLK (Vector-Optimized Library of Kernels) library is installed in
    $SDRROOT/dom/deps/gnuhawk/lib.
  • GNUHAWK components are installed in $SDRROOT/dom/components/gnuhawk. Within this directory, there are two subdirectories:
    • gr contains components that utilize GNU Radio blocks that have the prefix gr_.
    • digital contains components that utilize GNU Radio blocks that have the prefix digital_.
