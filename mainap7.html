Appendix G
REDHAWK Packages

Note: In the packages identified in this appendix, <dist> and <arch> correspond to the Linux distribution and architecture, respectively (For example, el6 and x86_64). <version> corresponds to the version of REDHAWK being installed.

The following REDHAWK packages are recommended for a standard install of the REDHAWK Framework for runtime-only use:

  • bulkioInterfaces-<version>.<dist>.<arch>.rpm
  • burstioInterfaces-<version>.<dist>.<arch>.rpm
  • frontendInterfaces-<version>.<dist>.<arch>.rpm
  • GPP-<version>.<dist>.<arch>.rpm
  • GPP-profile-<version>.<dist>.<arch>.rpm
  • jacorb-<version>.<dist>.<arch>.rpm
  • redhawk-sdrroot-dev-mgr-<version>.<dist>.<arch>.rpm
  • redhawk-sdrroot-dom-mgr-<version>.<dist>.<arch>.rpm
  • redhawk-sdrroot-dom-profile-<version>.<dist>.<arch>.rpm
  • redhawk-<version>.<dist>.<arch>.rpm
  • redhawk-ide-<version>.<dist>.<arch>.rpm

frontendInterfaces is versioned separately from REDHAWK.

The following additional packages are recommended for REDHAWK developers:

  • redhawk-basic-components-<version>.<dist>.<arch>.rpm
  • redhawk-basic-components-devel-<version>.<dist>.<arch>.rpm
  • redhawk-codegen-<version>.<dist>.<arch>.rpm
  • redhawk-devel-<version>.<dist>.<arch>.rpm

The following debugging symbol packages are provided:

  • bulkioInterfaces-debuginfo-<version>.<dist>.<arch>.rpm
  • burstioInterfaces-debuginfo-<version>.<dist>.<arch>.rpm
  • frontendInterfaces-debuginfo-<version>.<dist>.<arch>.rpm
  • redhawk-basic-components-debuginfo-<version>.<dist>.<arch>.rpm
  • redhawk-debuginfo-<version>.<dist>.<arch>.rpm
