Chapter 3
Installing REDHAWK on the BeagleBone Black

This chapter explains how to install REDHAWK on the BeagleBone Black by building from source.

Building REDHAWK on the BeagleBone may take several hours (approximately four).

To access the BeagleBone Black, ssh in from your development computer using the BeagleBone IP address assigned in Section 2.3. The default username is ubuntu and the default password is temppwd. For example, if the BeagleBone’s IP address is 192.168.1.125, you enter the following:

The following steps explain how to build and install REDHAWK on the BeagleBone Black.

You must execute these steps on the BeagleBone Black running the previously set up Ubuntu microSD card. For more information about setting up Ubuntu on the microSD card, refer to Section 2.2.

  1. Install the Dependencies.
  2. Edit /etc/hosts.
  3. Edit /etc/omniorb.cfg.
  4. Restart omniNames and omniEvents.
  5. Create a Symbolic Link for omniidl.
  6. Download and Install the REDHAWK framework.
  7. Download and Install BulkIO interfaces.
  8. Download and Install the GPP.
  9. Change Ownership of SDRROOT.

3.2 Edit /etc/hosts

Edit the /etc/hosts file so that omniNames works correctly. The new /etc/hosts file must read as follows:
127.0.0.1 localhost localhost.localdomain <ip_address of BeagleBone Black> <hostname of BeagleBone Black> localhost

For example, if the IP address is 192.168.1.125 and the hostname is arm, the second line reads as follows: 192.168.1.125 arm localhost

3.3 Edit /etc/omniORB.cfg

The omniORB configuration file (/etc/omniORB.cfg) on the BeagleBone Black must be edited to provide information about how to reach the CORBA event service. Edit /etc/omniORB.cfg by entering the following command and then making the following edits to the file.

  1. Delete line 377, DefaultInitRef = corbaloc::
  2. Add the line InitRef = NameService=corbaname::<ip_address of BeagleBone Black>
  3. Add the line InitRef = EventService=corbaloc::<ip_address of BeagleBone Black>:11169/omniEvents
