gov.redhawk. interceptor.client

Client Corba Interceptor

1.0.0

When enabled, intercepts CORBA invocation. It then validates if the invocation was made from a UI / Model locking context. This is to assist in DEBUGGING an application and making sure all CORBA classes don’t block critical threads. Refer to the readme within the plug-in for additional details.
