===================================================
REDHAWK CORE Framework v1.9.0 (released 2013-09-26)
===================================================

Fixed 183 bugs
- Added complex property types
- Application-level "uses device" relationship
- Application-level external properties
- Application-level external port renaming
- New component pattern; shifted the BulkIO implementation from generated code to base implementations
- New code generators
    - Easily-accessible command-line invocation of code generators
    - Python-based extension of code generator patterns
- Support for Ubuntu
- Matplotlib-based Python Sandbox plotting
- Unified (interoperable) Python Sandbox, live domain, and unit test packages
- Base class support for C++ and Java Service
