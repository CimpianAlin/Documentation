Appendix G
Known Compatibility Issues

G.1 Installing REDHAWK on RHEL/CentOS 6.5

The Java 7 OpenJDK RPMs shipped with RHEL/CentOS 6.5 contained a bug that prevents REDHAWK from installing via RPM. The issue does not affect building or installing REDHAWK from source. The issue was addressed by RedHat in later updates, and is corrected as of Java 7 update 51 (https://bugzilla.redhat.com/show_bug.cgi?id=1030634). Installing the latest updates for RHEL/CentOS 6.5 corrects the issue.

G.2 Compiling REDHAWK and REDHAWK C++
Components on RHEL/CentOS 6.4

The boost header files shipped with RHEL/CentOS 6.4 contained a bug that prevents REDHAWK and REDHAWK C++ projects from compiling. The issue was addressed by RedHat in later updates, and is corrected as of boost 1.41.0 release 17 (https://bugzilla.redhat.com/show_bug.cgi?id=921792). Installing the latest updates for RHEL/CentOS 6.4 corrects the issue.
