Appendix A
Eclipse Development

A.1 Eclipse Integrated Help

The REDHAWK IDE includes Eclipse integrated help. The following steps explain how to access the Eclipse integrated help within the IDE.

  1. From the Help menu, select Help Contents.
  2. The Help Contents dialog is displayed.

The Help Contents can be configured to open in the system web browser. From the REDHAWK IDE Window menu, select Preferences. In the Preferences dialog window, select Help and change the Open help search option to In a browser.
