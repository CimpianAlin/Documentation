2.3 Assigning a Static IP Address

Determining the dynamic IP address assigned to the BeagleBone Black is complicated and cumbersome; therefore, assigning the BeagleBone Black a static IP address is a best engineering practice. The following procedure enables you to assign a static IP address to the BeagleBone Black by editing the interfaces file on the microSD card prior to installing it on the BeagleBone Black.

  1. Enter the following command to create a directory on the development computer where the file system of the microSD card can be mounted.
  2. Enter the following command to mount the microSD card to the newly created directory.
  3. To assign the static IP address, edit /media/rootfs/etc/network/interfaces by entering the following command.
    1. On the line reading iface eth0 inet dhcp, replace dhcp with static.
    2. Add the following lines after iface eth0 inet static: address <desired ip address for BeagleBone Black> gateway <gateway for your network> netmask <netmask for your network> network <network prefix for your network = gateway AND netmask> broadcast <broadcast for your network> dns-nameservers 8.8.8.8

    An example of the changes to /media/rootfs/etc/network/interfaces is: iface eth0 inet static address 192.168.1.125 gateway 192.168.1.1 netmask 255.255.255.0 network 192.168.1.0 broadcast 192.168.1.255 dns-nameservers 8.8.8.8
