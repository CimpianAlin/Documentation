Chapter 2
Installation

2.1 Introduction

This chapter explains how to install the Core Framework, the IDE, and the Basic Components. The Core Framework is the software back-end of REDHAWK. The IDE is a Graphical User Interface (GUI) for development and use of REDHAWK systems. The Basic Components are building blocks that developers can use to create radio algorithms.

Complete installation of REDHAWK consists of the following steps:

  1. Installing Dependencies
  2. Installing the Core Framework
  3. Installing the IDE
  4. Setting Environment Variables
  5. Configuring omniORB

To configure and install REDHAWK and associated dependencies, you must have root permissions. It is recommended that users install from RPMs (available at http://redhawksdr.github.io/Documentation/download.html), but advanced users who desire custom functionality may also install from source. If installing from source, refer to Section D after installing dependencies.

REDHAWK installation is supported for Red Hat Enterprise Linux or Community Enterprise Operating System (CentOS) versions 5 and 6, for both 32-bit and 64-bit versions. A developer install of Red Hat Enterprise Linux or CentOS is recommended.

If you are running RHEL/CentOS 6.4, ensure the latest updates have been applied to your system. A bug was present in the initial release of 6.4 that causes compilation errors when developing for REDHAWK. If it is not possible for you to apply the latest updates, refer to Section J.

2.3 Installing Dependencies

REDHAWK has several open-source software dependencies, most of which are available from the basic Red Hat/CentOS repositories or from the Fedora Extra Packages for Enterprise Linux (EPEL) repository. Dependencies not available from either of those sources are included in the REDHAWK dependencies tarball. These dependencies are listed in Section C.1.

For more information on the Fedora EPEL project, refer to http://fedoraproject.org/wiki/EPEL.

2.5 Installing the IDE

The REDHAWK IDE leverages Java technologies, and for the best Java experience a minimum of 2GB of RAM is recommended. To install the REDHAWK IDE, you must unzip the appropriate installation file.

To develop and run Java-based components within the IDE, the environment variable JAVA_HOME must be set, and JAVA_HOME/bin must be added to the PATH. It is recommended that you put this in the .bashrc file. The location of JAVA_HOME depends on where Java is installed and is based on the system configuration.
