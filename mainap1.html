Appendix A
REDHAWK Packages

A.1 REDHAWK Packages

The following REDHAWK packages are recommended for a standard install of the REDHAWK Framework for runtime-only use:

  • bulkioInterfaces
  • burstioInterfaces
  • frontendInterfaces
  • GPP
  • GPP-profile
  • redhawk-sdrroot-dev-mgr
  • redhawk-sdrroot-dom-mgr
  • redhawk-sdrroot-dom-profile
  • redhawk

The following additional packages are recommended for REDHAWK developers:

  • redhawk-basic-components
  • redhawk-basic-devices
  • redhawk-basic-waveforms
  • redhawk-codegen
  • redhawk-devel
  • redhawk-ide
  • redhawk-qt-tools

The following debugging symbol packages are provided:

  • bulkioInterfaces-debuginfo
  • burstioInterfaces-debuginfo
  • frontendInterfaces-debuginfo
  • redhawk-basic-components-debuginfo
  • redhawk-debuginfo

A.2 Dependencies Packaged with REDHAWK

The following REDHAWK dependencies/support packages are included in the standard REDHAWK tarball for RHEL/CentOS 6:

  • libomniEvents2
  • libomniEvents2-devel
  • omniEvents-bootscripts
  • omniEvents-debuginfo
  • omniEvents-doc
  • omniEvents-server
  • omniEvents-utils
  • omniORBpy-debuginfo
  • omniORBpy-devel
  • omniORBpy-libs
  • python-omniORB
  • uhd
  • uhd-debuginfo
  • uhd-devel
  • uhd-doc
  • uhd-firmware

A.3 Dependencies From Fedora EPEL

The following REDHAWK dependencies/support packages for RHEL/CentOS 6 from EPEL are necessary for REDHAWK:

  • log4cxx
  • log4cxx-devel
  • omniORB
  • omniORB-devel
  • omniORB-doc
  • omniORB-servers
  • omniORB-utils
  • python-jinja2-26
  • xsd
  • xsd-doc
